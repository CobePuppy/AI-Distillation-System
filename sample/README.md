**离线版本（offline）运行**

1、从 huggingface 下载模型文件到 ./models 目录

[Qwen3-0.6B](https://huggingface.co/Qwen/Qwen3-0.6B/tree/main)、
[Qwen3-4B](https://huggingface.co/Qwen/Qwen3-4B/tree/main)、
[Qwen3-8B](https://huggingface.co/Qwen/Qwen3-8B/tree/main)、
[DeepSeek-R1-0528-Qwen3-8B](https://huggingface.co/deepseek-ai/DeepSeek-R1-0528-Qwen3-8B)

2、运行 offline.py 文件

**在线版本（online）运行**

1. 注册[阿里云百炼](https://bailian.console.aliyun.com/)账号，申请 API KEY

2. 用申请到的 API KEY 替换 online.py 中 api_key 参数

3. 运行 online.py 文件
